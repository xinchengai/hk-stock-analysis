# HK Stock Research Brief Format

Use this Markdown structure for each HK stock.

```markdown
## 港股研究简报：{name}（{ticker}）

数据时间：{fetch_time}  
数据来源：{data_source}  
结论口径：研究参考，不构成投资建议

### 1. 快速结论

| 项目 | 判断 |
|---|---|
| 研究倾向 | 偏强 / 中性 / 偏弱 |
| 核心驱动 | ... |
| 主要风险 | ... |
| 后续观察 | ... |

### 2. 行情与估值快照

- 最新价：{last_price} {currency}
- 涨跌幅：{change_pct}
- 成交量：{volume}
- 行情时间：盘中/收盘/截至 HH:MM，按来源原文填写
- 52周/阶段位置：如数据可得则填写
- 估值：如 PE/PB/股息率等可从可靠来源获得则填写；无法复算时标注“第三方口径，未能复算”

### 3. 技术面

- 趋势：结合 MA20/MA60/MA120 判断
- 动能：结合 MACD、RSI
- 量能：近期成交量相对均量
- 支撑/压力：基于近 180 日高低点与均线

### 4. 消息面与公告

- HKEX/公司公告：
- 公司新闻：
- 行业新闻：

### 5. 事实校验

| 校验项 | 结果 |
|---|---|
| HKEX 公告来源 | 公告标题、日期、链接；未取得则写“未取得可靠来源” |
| 行情时间口径 | 盘中/收盘/截至 HH:MM；盘中数据不得写收盘 |
| 涨跌幅复算 | (现价或收盘价 - 昨收) / 昨收 = ... |
| 折让/溢价复算 | (要约价或交易价 - 基准价) / 基准价 = ...；正数为溢价，负数为折让 |
| 财年口径 | 按 HKEX 年报/中报标题填写，例如“截至2025年4月30日止年度” |
| 估值口径 | 自行复算 / 第三方口径 / 未取得可靠来源 |
| 技术面可靠性 | 正常 / 事件驱动，技术面参考性降低 |

### 6. 市场与资金环境

- 恒生指数/恒生科技指数背景：
- 行业板块表现：
- 南向资金或资金面线索：如可得则填写

### 7. 证据权重

- 支持偏强的证据：
- 支持中性/偏弱的证据：
- 关键不确定性：

### 8. 观察清单

- 价格/技术观察：
- 财报/公告观察：
- 宏观/行业观察：
```
