# HK Stock Research Brief Format

Use this Markdown structure for each HK stock.

```markdown
## 港股研究简报：{name}（{ticker}）

数据时间：{fetch_time}  
数据来源：{data_source}  
结论口径：研究参考，不构成投资建议

### 1. 快速结论

| 项目 | 判断 |
|---|---|
| 研究倾向 | 偏强 / 中性 / 偏弱 |
| 核心驱动 | ... |
| 主要风险 | ... |
| 后续观察 | ... |

### 2. 行情与估值快照

- 最新价：{last_price} {currency}
- 涨跌幅：{change_pct}
- 成交量：{volume}
- 行情时间：盘中/收盘/截至 HH:MM，按来源原文填写
- 52周/阶段位置：如数据可得则填写
- 估值：如 PE/PB/股息率等可从可靠来源获得则填写；无法复算时标注“第三方口径，未能复算”

### 3. 流动性与港股市场特征

- 流动性：20日平均成交额、今日成交额、换手率、买卖价差；未取得则写“未取得可靠来源”
- 港股通/南向资金：是否港股通标的、南向持股/净流入；不适用或未取得则说明
- 卖空数据：卖空成交额、卖空占比、未平仓空头；未取得则说明
- AH 溢价：仅 A/H 双重上市公司填写；不适用则写“不适用”
- 港股交易特征：T+0、无涨跌停、流动性薄或事件驱动对波动的影响

### 4. 技术面

- 趋势：结合 MA20/MA60/MA120 判断
- 动能：结合 MACD、RSI
- 量能：近期成交量相对均量
- 支撑/压力：基于近 180 日高低点与均线

### 5. 行业专项分析

- 行业分类：
- 核心行业指标：按 `references/industry-frameworks.md` 选择，不可硬填
- 同业比较：如有可靠来源则填写
- 行业风险：

### 6. 事件驱动与公告

- HKEX/公司公告：
- 公司新闻：
- 行业新闻：
- 事件类型：要约 / 配售 / 回购 / 分红 / 指数变动 / 港股通变动 / 并购 / 复牌 / 其他
- 事件影响：说明短期催化、长期基本面影响和不确定性

### 7. 事实校验

| 校验项 | 结果 |
|---|---|
| HKEX 公告来源 | 公告标题、日期、链接；未取得则写“未取得可靠来源” |
| 行情时间口径 | 盘中/收盘/截至 HH:MM；盘中数据不得写收盘 |
| 涨跌幅复算 | (现价或收盘价 - 昨收) / 昨收 = ... |
| 折让/溢价复算 | (要约价或交易价 - 基准价) / 基准价 = ...；正数为溢价，负数为折让 |
| 财年口径 | 按 HKEX 年报/中报标题填写，例如“截至2025年4月30日止年度” |
| 估值口径 | 自行复算 / 第三方口径 / 未取得可靠来源 |
| 技术面可靠性 | 正常 / 事件驱动，技术面参考性降低 |
| 流动性口径 | 成交额/换手率/价差/自由流通股来源；未取得则说明 |
| 条件模块 | AH溢价/南向/卖空/行业指标是否适用 |

### 8. 市场与资金环境

- 恒生指数/恒生科技指数背景：
- 行业板块表现：
- 南向资金或资金面线索：如可得则填写

### 9. 证据权重

- 支持偏强的证据：
- 支持中性/偏弱的证据：
- 关键不确定性：

### 10. 观察清单

- 价格/技术观察：
- 财报/公告观察：
- 宏观/行业观察：
- 流动性/卖空/南向观察：
```
